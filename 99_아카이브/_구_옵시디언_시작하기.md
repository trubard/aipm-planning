# 👋 시작하기

이 폴더는 **옵시디언 볼트**입니다. 안에 들어있는 건 전부 그냥 마크다운(`.md`) 파일이에요.

## 폴더 구조

| 폴더 | 용도 |
|---|---|
| `00-Inbox` | 일단 떠오르는 거 아무거나 던져두는 곳 |
| `01-Notes` | 정리된 일반 노트 |
| `02-검색트렌드` | 수집한 검색 키워드 아카이브 |
| `03-리서치` | 리서치 결과·자료 정리 |
| `99-Templates` | 노트 템플릿 |

## 클로드 코드랑 쓰는 법

이 볼트 폴더는 클로드가 직접 읽고 쓸 수 있습니다. 예를 들어:

- "오늘 검색 트렌드 수집해서 `02-검색트렌드`에 노트로 저장해줘"
- "이 회의록 정리해서 볼트에 넣고 관련 노트랑 `[[백링크]]`로 연결해줘"
- "`03-리서치`에 있는 노트들 요약해줘"

## 옵시디언으로 여는 법

1. [obsidian.md](https://obsidian.md) 에서 옵시디언 설치
2. 앱 실행 → **"Open folder as vault"** → 이 폴더(`vault`) 선택
3. 끝. 노트끼리 `[[ ]]` 로 연결하면 그래프로 보입니다.
